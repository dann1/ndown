<h1>ndown</h1>

<h2>Required software</h2>

Install aria2, wget and youtube-dl

<h3>Ubuntu</h3>
```bash
sudo apt install aria2 wget youtube-dl
```
<h3>Archlinux </h3>
```bash
sudo pacman -S aria2 wget youtube-dl
```

<h2>crontab</h2>

Create a crontab for download schedule

```bash
crontab -e
```
